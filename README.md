# rclp-cli

rclp CLI.


## How to install

```
$ npm i rclp-cli
```

## How to use

Install it first, then authenticate using your Google Account:

```
$ rclp -a
```

Paste the latest data from your remote clipboard:

```
$ rclp -p
```

Copy data to your remote clipboard from stdin:

```
echo "hi there" | rclp -c
```
